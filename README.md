# MPOnline Advanced Software Engineering Assignments

A repository containing C# (.NET), SQL, and study notes assignments.

## Contents
- **C# Console Applications** (`02_drone_system` to `21_number_guessing`, `24_fizzbuzz_variant`): Demos of OOP concepts, Exceptions, LINQ, custom sorting, Collections, and basic logic.
- **SQL Projects** (`01_sql_users`, `13_sql_customer_product_order`, `22_sql_sales`, `23_sql_shipments`): Scripts for tables, joins, aggregations, and query optimization.
- **EF Core App** (`25_expense_sharing_efcore`): An expense sharing system backed by SQLite.
- **Study Notes** (`08_notes_collection_framework`, `09_notes_object_methods`, `26_homework_notes`): Summary notes on C# concepts, SDLC, ORM, and database design.

## How to Run
- **C# (Simple)**: Copy the code from `Program.cs` to [dotnetfiddle.net](https://dotnetfiddle.net/) or create a new local project using `dotnet new console`.
- **SQL**: Execute the self-contained `.sql` scripts in any standard SQL server client (e.g., SSMS, DBeaver).
- **EF Core**:
  ```bash
  cd 25_expense_sharing_efcore
  dotnet run
  ```
